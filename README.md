# README

## Ruby API

* clone do projeto
```git 
$ git clone https://github.com/Pedrogregorio/ruby-api/
```

* instalar o rvm
* instalar dependencias
```git
$ bundle install
```
* configurar o Arquivo de banco de dados
```
config/database.yml

$ rails db:create db:migrate
```
* rodar o projeto
```git
$ rails s
```
